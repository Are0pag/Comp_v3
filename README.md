Написана большая часть масштабного проекта на WPF.
Стек: C#, .NET, WPF, MVVM, DataBindings, EFCore, SQLite, Repository pattern

Есть собственные прикладные наработки: расширение Command паттерна до транзакцинной модели поведения с Fluent-интерфейсом, включая комбинацию нескольких команд в одну, отложенные команды.

Проект представляет собой складскую программу учёта процесса производства радиоэлектронной аппаратуры. Функционал включает:
- каталогизацию (TreeView, DataGrid), формирование составов сборочных единиц, справочников компонентов;
- документирование производственного процесса (поступления, сборка, учет аналогов компонентов, учет отгрузки готовой продукции);
- инструменты для ведения финансовой отсчётности, взаимодействия с поставщиками и клиентами (закупочные ведомости, оформление заказов покупателей, счетов на оплату.
